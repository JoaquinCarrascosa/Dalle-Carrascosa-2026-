# Dalle-Carrascosa-2026-
Dataset and replication code for Dalle & Carrascosa (2026). Can Cumulative Disadvantages Be Reversed? Class Attainment and a Network Analysis of Intergenerational Occupational Pathways by Migratory Origin in Buenos Aires, Argentina.
Dataset includes data from two probabilistic surveys (2021 and 2023). 

-2021. PISAC-COVID 2021 survey on “Social structure and public policy from Argentina” (using data only for the MABA)

-2023. IIGG-UBA/UNGS 2023 survey on “Occupational trajectories and class formation”.
